# Popup

Base code :

```
<div class"popupWrapper" id="idPopup">
    <div class="popup">
        <div class="wrap">
            <h3>Contoh Popup</h3>
        </div>
    </div>
</div>
```

### Memunculkan Popup

```js
munculPopup("#idPopup");
```

### Menghilangkan Popup

```js
hilangPopup("#idPopup");
```
